# What Really Happened (WRH)

## Overview

What Really Happened (WRH) is a structured, facilitator-led trauma education system designed to support individuals experiencing Complex Post-Traumatic Stress (C-PTSD).

WRH is intended for use in **institutional, community, and veteran-serving environments** as a structured psychoeducational program that complements existing clinical care.

---

## Purpose

WRH translates trauma science into a **repeatable, session-based learning framework** that supports:

- Nervous system regulation awareness
- Trauma pattern recognition
- Emotional stabilization skills
- Reduced shame-based identity structures
- Improved relational safety and communication

WRH is not a clinical treatment replacement. It is a structured educational support system designed to operate alongside licensed care when appropriate.

---

## System Structure

WRH is organized into four core components:

- **Clinical Curriculum** → Facilitator-led 12+ session framework
- **Safety Systems** → Decompression, crisis awareness, stabilization tools
- **Implementation Framework** → How organizations deploy WRH
- **Measurement Layer** → Pre/post outcomes and qualitative feedback

---

## Intended Use Settings

WRH is designed for:

- Veteran support programs
- Community mental health organizations
- Group-based trauma education settings
- Justice-involved rehabilitation programs
- Nonprofit behavioral health initiatives

---

## Safety Notice

WRH is not appropriate as a standalone intervention for:

- Active psychosis
- Acute suicidal crisis without clinical oversight
- Unmanaged severe dissociation requiring stabilization
- Situations requiring inpatient psychiatric care

All implementations must include appropriate escalation pathways.
