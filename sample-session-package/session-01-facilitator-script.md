# Session 01 Facilitator Script (Sample)

## Overview

Session 01 introduces the core WRH reframe: survival responses are adaptations, not character defects.

## Who it is for

Lead/co-facilitators delivering WRH Session 01.

## How to use

1. Deliver the cold open.
2. Run quick grounding.
3. Teach core concept with pacing.
4. Facilitate mirror moment without forced disclosure.
5. Close with open-loop prompt.

## Script blocks (short form)

- Cold open: "Your system learned survival before it learned trust."
- Grounding: "Feet on floor, one slow breath out."
- Core teaching: developmental threat -> adaptation patterns
- Mirror moment: identify one moment of shutdown/freeze and reframe
- Close: notice one survival behavior this week without judgment

## Limits and disclaimers

Do not process detailed trauma narratives in this session; redirect to individual clinical support as needed.
