# WRH Evidence Base

## Overview

WRH is a trauma psychoeducation program informed by established trauma frameworks and translated into lived-experience language for group learning settings.

## Who it is for

Program designers, facilitators, and partner organizations that need a clear rationale for how WRH session content aligns with trauma-informed concepts.

## How to use

- Use this document to explain *why* each session exists.
- Pair with `facilitator-manual.md` for practical delivery.
- Pair with `OUTCOMES_AND_MEASURES.md` for evaluation.

## What WRH is and is not

- **WRH is:** structured trauma psychoeducation that helps participants identify patterns, increase regulation skills, and reduce shame through understanding.
- **WRH is not:** psychotherapy, diagnostic assessment, or crisis stabilization.

## Session-to-framework mapping

| Session | Core concepts | Why it matters |
|---|---|---|
| Session 00 (Anchor) | Safety cues, orientation, stabilization | Establishes predictability and lowers activation before content exposure |
| Session 01 (Blueprint) | Developmental trauma, survival adaptation, attachment threat | Reframes "what is wrong with me" into adaptation under threat |
| Session 02 (Masks) | Fawn response, overfunctioning, boundary collapse | Helps participants name appeasement as survival strategy |
| Session 03 (Love Hurts) | Intermittent reinforcement, trauma bonding, neuroception | Clarifies why dangerous relationships can feel familiar/safe |
| Session 04 (War on Reality) | Gaslighting effects, self-trust erosion, cognitive dissonance | Rebuilds reality testing and language for coercive dynamics |
| Session 05 (Relapse) | Shame cycles, affect regulation, coping substitution | Reduces moral injury framing and supports behavior-change planning |
| Session 06 (Healing) | Co-regulation, pacing, embodied regulation | Moves from insight to repeatable regulation behaviors |
| Session 07+ | Grief integration, identity repair, narrative reconstruction | Supports long-term recovery through coherent self-understanding |

## Frameworks used (plain-language)

### 1) Polyvagal-informed lens
- Focuses on autonomic state shifts (safety, mobilization, shutdown) and how social cues change regulation capacity.
- WRH uses short grounding and pacing cues to support state awareness in-session.

### 2) Window of tolerance
- Teaches participants to recognize hyperarousal/hypoarousal and return toward workable activation.
- WRH embeds micro-regulation pauses and "recognition without overwhelm" sequencing.

### 3) Parts-work / IFS-adjacent principles
- Uses non-pathologizing language to describe conflicting survival parts (e.g., protector behaviors).
- WRH avoids forced disclosure and instead emphasizes curiosity, self-observation, and choice.

### 4) Somatic regulation principles
- Uses body-based anchors (breath, posture, orienting) to reduce cognitive overload and improve retention.
- WRH repeats brief somatic check-ins every session to normalize regulation practice.
